# 🌌 Dark Matter Classification System

An intelligent dark matter event classification and anomaly detection platform leveraging AI-driven reasoning for scientific data analysis and research insights.

---

## 🚀 Overview

This system enables automated classification of dark matter events and detection of anomalous patterns in astrophysical datasets using advanced AI models.

It provides a full-stack research interface including real-time dashboards, batch processing capabilities, and AI-assisted scientific reasoning.

---

## ✨ Key Features

- 🔬 AI-Powered Event Classification  
- ⚠️ Intelligent Anomaly Detection  
- 📊 Real-time Results Dashboard  
- 📈 Dataset Analytics & Visualization  
- 📁 Batch Upload & Processing  
- 🤖 Claude AI Integration for scientific reasoning  

---

## 🏗️ System Architecture

Dataset → Flask API → Classification Engine → Anomaly Detection Module → AI Reasoning Layer → React Dashboard
